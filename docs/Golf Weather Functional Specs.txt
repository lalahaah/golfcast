⛳️ 골프 웨더(Golf Weather) 초상세 기능 요구사항 명세서 (FRD v1.0)1. 문서 개요본 문서는 '골프 웨더' MVP 개발을 위한 기능적/비기능적 요구사항을 상세히 정의한다. 개발자는 본 문서를 기준으로 데이터 구조 설계, API 연동, UI 컴포넌트 개발을 진행한다.2. 프론트엔드 UI/UX 상세 명세2.1. 메인 (Home) & 검색 기능목표: 사용자가 진입 후 3초 이내에 원하는 골프장을 찾게 한다.진입 화면 (Initial State):헤더: 로고(GolfCast), 공유 버튼(우측 상단).중앙: "어디로 라운딩 가시나요?" 메인 카피.검색창: 화면 중앙에 위치. autofocus 속성 적용 (모바일에서는 키패드 자동 오픈).배경: 깔끔한 단색 배경(Slate-50) 또는 매우 흐릿한 골프장 전경 이미지 (Lazy Loading 적용).검색 인터랙션 (Search Interaction):Debounce: 사용자 입력 시 API 과부하 방지를 위해 300ms 지연 후 검색 로직 실행.검색 로직:입력값: searchTerm (예: "레이크")대상: 골프장 DB의 name_kr, name_en, aliases 필드.매칭 방식: includes() (부분 일치). 초성 검색은 MVP 제외 (추후 도입).결과 리스트 (Dropdown):검색창 바로 하단에 리스트 형태로 노출.최대 5~7개 항목 표시. 스크롤 가능.각 항목 구성: 골프장 이름 (Bold), 지역 (예: 용인, 가평 - Gray text).No Result: 검색 결과가 없을 경우 "검색 결과가 없습니다. 골프장 이름을 확인해주세요." 문구 노출.2.2. 날씨 상세 페이지 (Detail View)목표: 스크롤 없이 핵심 정보(가부 결정)를 확인하고, 스크롤 하여 세부 전략을 짠다.데이터 로딩 (Loading State):Skeleton UI: 데이터가 로드되는 1~2초 동안 텍스트와 카드 위치에 회색 박스 애니메이션(Pulse) 표시. (사용자 이탈 방지 핵심)섹션 1: 종합 스코어 카드 (The Verdict)위치: 최상단.골프 지수 (Score): 0~100점 숫자.컬러 코딩:80~100점: Text-Green-600, 배경 Bg-Green-50 (아이콘: 웃는 얼굴/트로피)50~79점: Text-Yellow-600, 배경 Bg-Yellow-50 (아이콘: 평범한 얼굴/골프채)0~49점: Text-Red-600, 배경 Bg-Red-50 (아이콘: 우산/경고)메인 멘트: 점수에 매핑된 한 줄 요약 (예: "라베 도전하기 완벽합니다!").섹션 2: 타임라인 (Hourly Forecast)표시 시간: 현재 시간 기준 + 12시간.UI 형태: 가로 스크롤(Horizontal Scroll) 가능한 카드 리스트.카드 구성: 시간 (예: 09:00), 날씨 아이콘, 기온, 풍속/강수량.강조: '오전 6시 ~ 오후 2시' (주요 티오프 시간)는 배경색을 미세하게 다르게 하여 강조.섹션 3: 4대 요소 상세 분석 (Grid Layout)2x2 그리드 레이아웃.바람: 풍속(m/s) + 풍향(화살표 아이콘). 5m/s 이상 시 텍스트 빨간색 강조.기온/체감: 실제 기온 vs 체감 온도.강수/습도: 강수 확률(%) 및 습도. 습도 90% 이상 시 "안개 주의" 뱃지 부착.자외선(UV): 맑은 날 필수 표시. "선크림 필수" 등의 팁 제공.섹션 4: AI 캐디 조언 (Text Generation)말풍선 UI 디자인.조건문(If-else) 로직에 의해 조합된 문장 출력.3. 백엔드 및 데이터 로직 상세3.1. 골프장 데이터 구조 (JSON Schema)확장성을 고려하여 aliases(별칭)과 region(지역)을 분리한다.[

3.2. 알고리즘: 골프 지수 (Golf Weather Index) 산출 공식개발자는 아래 공식을 코드로 구현한다.기본 점수: 100점 만점에서 감점 방식 (Deduction System)바람 감점 (Wind Penalty):0 ~ 3m/s: 감점 없음4 ~ 7m/s: -10점 (약간 영향)8 ~ 11m/s: -25점 (클럽 선택 영향 큼)12m/s 이상: -40점 (플레이 어려움)강수 감점 (Rain Penalty):강수량 0mm: 감점 없음시간당 0.1 ~ 1mm (이슬비): -15점시간당 1 ~ 5mm (보통 비): -30점시간당 5mm 이상 (호우): -50점 (경기 중단 가능성)기온 감점 (Temp Penalty):5°C ~ 28°C: 감점 없음 (골든 존)0°C ~ 4°C 또는 29°C ~ 32°C: -10점영하(<0°C) 또는 폭염(>33°C): -30점최종 계산: Score = Max(0, 100 - WindP - RainP - TempP)3.3. API 연동 및 에러 처리 (OpenWeatherMap)Endpoint: /data/3.0/onecall (유료이나 하루 1,000건 무료)Parameters: lat, lon, exclude=minutely,daily, units=metric, lang=krException Handling:API 호출 실패 (500/Timeout): "기상청 연결 상태가 좋지 않습니다. 잠시 후 다시 시도해주세요." 토스트 메시지 출력.데이터 누락: 특정 시간대 예보가 없을 경우 가장 가까운 시간대 데이터로 대체(Fallback).4. 비기능 요구사항 (Non-Functional Requirements)4.1. 성능 (Performance)LCP (Largest Contentful Paint): 2.5초 이내 목표. (이미지 최적화 필수)CLS (Layout Shift): 날씨 데이터 로딩 전/후 레이아웃이 덜컥거리지 않도록 min-height를 지정한다.4.2. SEO (검색 엔진 최적화)Dynamic Meta Tags: 골프장별로 title과 description이 동적으로 바뀌어야 한다.Title: "레이크사이드 CC 내일 날씨 및 골프 지수 | GolfCast"Desc: "레이크사이드 CC 내일 비 올까요? 풍속 3m/s, 라운딩 지수 90점. 옷차림 추천까지 확인하세요."Open Graph (OG): 카카오톡 공유 시 썸네일과 요약 정보가 정확히 떠야 한다.4.3. 호환성 (Compatibility)Mobile First: 모든 UI는 모바일(iPhone SE 너비 320px) 기준으로 먼저 설계하고 태블릿/PC로 확장한다.Cross Browser: Chrome, Safari (iOS), Samsung Internet 지원.5. 데이터 수집 및 관리 계획초기 데이터: 구글 맵스 API 또는 수작업으로 수도권 Top 50 골프장 좌표 확보 (JSON 파일).업데이트: 사용자가 "여기 골프장 없어요" 제보 기능을 통해 요청하면 개발자가 JSON 업데이트 후 재배포.
