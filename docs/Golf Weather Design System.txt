🎨 GolfCast Design System: "The Green Code" (v1.0)1. 디자인 원칙 (Design Principles)1.1. Glanceability (0.1초의 인지)골퍼는 필드 위에서, 혹은 운전 중에 앱을 봅니다.원칙: 텍스트를 읽게 하지 말고, 색상과 아이콘으로 '느끼게' 합니다.점수가 80점이면 화면 전체가 초록빛이어야 하고, 40점이면 붉은빛이어야 합니다.1.2. Thumb-Friendly (엄지 중심 설계)주 타겟층(3050)은 한 손 조작을 선호합니다.원칙: 모든 핵심 액션(검색 결과 클릭, 공유하기)은 화면 하단 2/3 영역 안에서 해결합니다.1.3. Scalable Neutrality (확장 가능한 중립성)날씨 정보가 주인공입니다. UI는 배경으로 물러나야 합니다.원칙: 화려한 그라데이션을 배제하고, Slate 계열의 차분한 중립 컬러를 베이스로 사용합니다.2. 컬러 팔레트 (Color System)Tailwind CSS 표준 컬러를 기반으로, 의미론적(Semantic) 이름을 부여하여 관리합니다.2.1. Brand & Functional Colors (The Traffic Light)점수(Score)와 날씨 상태를 직관적으로 전달하는 핵심 컬러입니다.RoleSemantic NameTailwind ColorHex CodeUsagePrimaryBrand Greengreen-700#15803d로고, GNB 헤더, 주요 버튼PositiveSignal Greenemerald-500#10b98180점 이상 (Good), 맑음, 추천CautionSignal Yellowamber-400#fbbf2450~79점 (So-so), 흐림, 주의NegativeSignal Redrose-500#f43f5e49점 이하 (Bad), 비/눈, 경고InfoSky Bluesky-500#0ea5e9일반 정보, 링크, 바람(약)2.2. Type Scale모바일 가독성을 위해 기본 폰트 사이즈를 넉넉하게 잡습니다.Display XL (Score): 64px (Bold) - 상세 화면의 골프 점수Heading 1: 24px (Bold) - 메인 타이틀Heading 2: 20px (SemiBold) - 섹션 타이틀 (예: 시간별 날씨)Body 1: 16px (Medium) - 일반 텍스트, 검색 결과Body 2: 14px (Regular) - 보조 설명Caption: 12px (Regular) - 태그, 날짜, 저작권4. UI 컴포넌트 라이브러리 (Components)4.1. Cards (Container)이 앱의 모든 정보는 '카드'에 담깁니다.Shape: Rounded-2xl (16px) - 부드럽고 현대적인 느낌.Shadow: Shadow-sm - 얕은 그림자로 떠 있는 느낌만 줌.Padding: p-4 (16px) 또는 p-6 (24px).4.2. Icons (Visual Language)Library: Lucide React (가볍고 선이 깔끔함).Size:Small: 16px (부가 정보)Medium: 24px (일반 아이콘)Large: 32px ~ 48px (날씨 상태 아이콘)Style: StrokeWidth 2px. 둥근 끝처리(Round Join).4.3. Skeleton UI (Loading)사용자가 데이터를 기다리는 시간을 짧게 느끼게 합니다.Animation: Pulse (부드럽게 깜빡임).Color: bg-slate-200.Shape: 실제 콘텐츠와 동일한 크기의 Rounded Box.5. 레이아웃 & 그리드 (Layout System)5.1. Spacing (8pt Grid)모든 간격은 8의 배수로 정의하여 일관성을 유지합니다.gap-1 (4px): 아주 가까운 요소 (텍스트와 작은 아이콘)gap-2 (8px): 연관된 요소 그룹gap-4 (16px): 카드 내부 섹션 간격gap-6 (24px): 큰 섹션 간격5.2. Responsive ContainerMax Width: max-w-md (448px)Alignment: mx-auto (중앙 정렬)Explanation: 데스크탑에서 열어도 모바일 앱처럼 보이도록 좌우 여백을 두고 중앙에 배치합니다.6. 마이크로 인터랙션 (Micro-Interactions)6.1. 검색 결과 피드백검색어 입력 시 리스트가 Fade-in + Slide-down으로 나타납니다.항목 터치 시 배경색이 bg-slate-100으로 즉각 변경(Active State)되어 눌렸음을 인지시킵니다.6.2. 점수 노출 (Wow Effect)상세 페이지 진입 시, 골프 점수(숫자)가 0에서 85까지 빠르게 카운트 업(Count-up) 되며 사용자에게 기대감을 줍니다.
